# Getting Access

At the Simons Foundation, Claude Team is provisioned through IT.

1. Go to [myaccess.microsoft.com](https://myaccess.microsoft.com/)
2. Sign in with your `@flatironinstitute.org` email
3. Request access to **Anthropic Claude**

You'll get a **standard** Team seat by default.

---

## Hitting Usage Limits?

Request upgrade from **standard** to **premium**

✉️ [it@simonsfoundation.org](mailto:it@simonsfoundation.org)
