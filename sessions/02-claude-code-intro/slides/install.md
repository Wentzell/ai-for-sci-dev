# Installing Claude Code

**Recommended:** native installer

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

- Auto-updates on the `latest` channel · `claude update` to update manually

On first launch, run `/login` and pick **simonsfoundation**.

Walk-through: [code.claude.com/docs/en/quickstart](https://code.claude.com/docs/en/quickstart)

Note:
Buffer ~1 minute here for stragglers. If anyone hits node-version or quota issues with npm/Homebrew, point them at the curl installer.
