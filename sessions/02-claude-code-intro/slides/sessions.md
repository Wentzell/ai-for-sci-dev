# Starting & Resuming Sessions

<div class="two-col">
<div>

**From the shell**
- `claude` ‚Äî new session here
- `claude -c` ‚Äî continue most recent
- `claude -r` ‚Äî picker across past sessions
- `claude -p "..."` ‚Äî one-shot for scripts

</div>
<div>

**Mid-session keys**
- `Esc` ‚Äî stop mid-response
- `Esc Esc` ‚Äî rewind / undo to a prior turn (or `/rewind`)
- `/resume` ¬∑ `/fork` ¬∑ `/rename` ‚Äî branch & navigate
- `/clear` ‚Äî drop history

</div>
</div>

Note:
`Esc` (single) is muscle memory worth building ‚Äî most "oh no, stop" moments are solved by it. The `Undo anything` `/powerup` lesson goes deeper into rewind.
