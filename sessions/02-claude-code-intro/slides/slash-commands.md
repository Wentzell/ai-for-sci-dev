- `/config` — settings UI
- `/context` — token usage breakdown
- `/usage` — cost & usage stats
- `/permissions` — view & edit rules

**Tune the model**
- `/model` — pick mid-session *(Sonnet ↔ Opus)*
- `/effort` — dial reasoning effort
